Members:
  Josh Achorn
  Brian Satchell

The program is a car rental simulator. 
* Keeps track of which cars in inventory and records of each day. Runs for 35 days and 34 nights. 
* returns cars that are due at beginning of day, outputs cars in shop, cars currently rented, and cars returned that day
* Picks Random number of customers for each day and once every customer has been helped it will move onto the next day. 

Object Oriented Patterns used:
* strategy pattern (customer class)
* Factory Pattern (car class)
* Decorator patter (car class)

output is going to be sim.out
- should have all the info of what went on each day 

To run just run the simulator file.
